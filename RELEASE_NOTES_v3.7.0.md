# StatusOS v3.7.0 – Performance Timer

## Added
- Focus, Tabata, Boxing, HIIT, EMOM and Stopwatch timer modes.
- Classic Tabata, Boxing, Kettlebell and Skipping presets.
- Work, rest, rounds, warm-up and cool-down controls.
- Bell 1 for work, Bell 2 for rest and halfway, Bell 3 for workout completion.
- Bell previews, vibration and optional spoken cues.
- Phase colours, round display, workout history, total time, total rounds and workout streak.

No new Supabase migration is required. Workout history is stored locally in this release.
