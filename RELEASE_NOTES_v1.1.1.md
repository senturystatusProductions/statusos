# StatusOS v1.1.1 – UX Polish

## Fixed
- Modal X buttons now close reliably on desktop and iPhone.
- Users are no longer trapped inside Add Opportunity or other entry dialogs.

## Added
- Visible Cancel buttons on all standard data-entry dialogs.
- Escape-key closing on desktop.
- Tap/click outside a dialog to close it.
- Larger mobile-friendly close targets.
- Automatic focus on the first available dialog control.
- `StatusOS.Modal` internal API for future dialogs.

## Preserved
All existing authentication, cloud sync, Mission Control, Task Engine, Habit Engine, Music OS, Daily Briefing, AI, CRM, and developer tools remain unchanged.
