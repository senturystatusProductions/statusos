SENTURY STATUS BUSINESS OS v2
=============================

WHAT THIS VERSION INCLUDES
- Daily dashboard and checklist
- Top 3 editable priorities
- Weekly routine by day
- Artist CRM
- Content planner
- Sales pipeline
- Project tracker
- Revenue tracker
- Goal tracker
- DM template library
- Export and import backups
- Automatic daily reset
- Offline support after first visit
- iPhone Home Screen installation
- Desktop and mobile responsive design

IMPORTANT LIMITATION
This version saves data locally in the browser on each device.
It does not yet include a true login, cloud syncing, or an AI assistant.
Those features require an online database/backend service such as Supabase or Firebase.

HOW TO PREVIEW ON WINDOWS
Double-click index.html.
Most features will work, but offline installation requires web hosting.

HOW TO PUBLISH FOR FREE WITH NETLIFY
1. Go to Netlify Drop in a browser.
2. Drag the entire SenturyStatusOS_v2 folder onto the upload page.
3. Netlify will give you a website address.
4. Open that address on your iPhone in Safari.
5. Tap Share.
6. Tap Add to Home Screen.
7. Name it Sentury Status OS and tap Add.

HOW TO HOST ON YOUR OWN WEBSITE
Upload every file in this folder to the same public folder, for example:
senturystatus.com/os/

Then open:
https://senturystatus.com/os/

BACKUPS
Use Export Backup regularly.
This downloads a JSON file containing all app data.
Use Import Backup to restore it.
