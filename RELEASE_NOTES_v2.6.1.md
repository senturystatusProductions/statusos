# StatusOS v2.6.1 — Quiet Sync & Pull to Refresh

## Quiet background sync
- Automatic task and Artist OS sync success messages no longer appear as pop-up notifications.
- The small sync indicator still reflects background state.
- Offline, setup, and genuine error notices remain visible.
- Manual **Sync Now** displays one clear confirmation when finished.

## iPhone pull to refresh
- In the Home Screen app, pull down from the very top of a page.
- StatusOS displays **Release to refresh** once the refresh threshold is reached.
- Releasing performs a quiet data refresh and reloads the app.
- The old pull-down gesture no longer opens the navigation menu. Use **More** for the full menu.

No SQL changes are required. Existing data and backups remain unchanged.
