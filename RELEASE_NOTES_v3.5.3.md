# StatusOS v3.5.3 — Focus Alarm Studio

- Added six user-recorded alarm sounds plus the original StatusOS chime.
- Added sound previews, alarm volume, sound on/off, and vibration controls.
- Alarm now plays for 30 seconds, 60 seconds, or until Stop Alarm is pressed.
- Added a large Stop Alarm completion panel with session totals.
- Added 10-second and 30-second test presets.
- Added custom minutes and seconds input.
- Added weekly, monthly, yearly, streak, and best-day focus statistics.
- Preserved Focus Planner cloud sync and Pomodoro bubble reset behavior.
- No new Supabase SQL is required.
