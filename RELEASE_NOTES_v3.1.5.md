# StatusOS v3.1.5 — Design System & UX Polish

## What changed

- Introduced shared spacing, radius, touch-target, card, button, form, and focus-state rules.
- Simplified the iPhone top bar and moved secondary utilities into the profile menu.
- Reworked the iPhone bottom navigation around Home, Today, Artists, Projects, and More.
- Turned the full mobile navigation into a cleaner elevated sheet.
- Reduced dashboard density on smaller screens.
- Standardized mobile grids, headers, forms, timeline cards, project layouts, and touch controls.
- Preserved all existing records, sync behavior, modules, and project data.
- Updated visible, internal, and service-worker versions to v3.1.5.

## Installation

Replace the previous StatusOS files with this release, then fully close and reopen the iPhone Home Screen app so the new cache activates.

No SQL update is required.
