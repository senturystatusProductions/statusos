# StatusOS v2.8.0

## Streamlined iPhone and Artist OS
- Reduced mobile dashboard and Artist OS visual clutter.
- Artist directory and artist profile now behave as separate mobile screens.
- Added a clear All Artists back button.
- Moved secondary artist contact fields into a collapsible details section.
- Simplified mobile artist statistics and activity entry layout.
- Preserved all existing artist records and sync behavior.

## Daily Mindset
- Added a morning and night routine inspired by the central principles presented in Think and Grow Rich.
- Morning: definite purpose, faith, visualization, gratitude, and one committed action.
- Night: wins, lesson, evidence of progress, tomorrow's priority, and pre-sleep visualization.
- Added daily completion storage and a full-day streak counter.
- No SQL migration required.
