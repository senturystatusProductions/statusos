# StatusOS v2.5.0 Smart Task Engine

- Rich task records with priority, status, estimated duration, progress, due date, project, artist, category, notes, dependencies, and recurrence.
- Search and filters.
- Workload, due-today, open, and completed metrics.
- Start/pause and complete/reopen actions.
- Completing a recurring task creates the next occurrence.
- Existing tasks are upgraded automatically with safe defaults.
- Core task fields continue syncing with the existing cloud table; rich fields remain protected in the local-first record in this release.
