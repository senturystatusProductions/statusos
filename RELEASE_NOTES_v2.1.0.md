# StatusOS v2.1.0 — Workspace Engine

## Added
- Restores the exact StatusOS module after refresh or reopening the app.
- Restores the open navigation group and expanded dashboard panels.
- Restores the selected Artist OS artist.
- Restores Artist OS search and status filters.
- Restores scroll position separately for each module.
- Saves workspace state automatically during navigation and editing.

## Notes
- Workspace state is stored locally on each device for instant and reliable recovery.
- Artist records continue to sync through Supabase as before.
- No SQL migration is required for this release.
