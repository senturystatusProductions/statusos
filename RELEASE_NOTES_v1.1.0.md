# StatusOS v1.1.0 – Daily Briefing

## Added
- Live Daily Briefing card in Mission Control.
- Recommended work order based on tasks, urgent music deadlines, and incomplete habits.
- Task, habit, and Music OS status summary.
- Context-aware daily insight.
- Manual refresh and automatic refresh whenever core data changes.
- `StatusOS.Briefing` internal API.

## Compatibility
No new Supabase migration is required. Existing cloud-synced data is read through the established services.
