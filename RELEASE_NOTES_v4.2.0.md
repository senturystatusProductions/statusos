# StatusOS v4.2.0 — Mission Control

## Added
- New Mission Control home section with Business Score and daily guidance.
- Today's Mission automatically ranks the three highest-impact open tasks.
- Continue Working card surfaces the most recently active project.
- Live session, revenue, open-task, overdue-task, and active-project metrics.
- Session Summary powered by the v4.1 Session Engine.
- Today's Timeline for completed sessions, completed tasks, and revenue entries.
- Quick Actions for tasks, artists, projects, sessions, and AI.
- Daily Win capture stored locally for long-term progress history.

## Compatibility
- Preserves the existing dashboard, Artist CRM, Command Palette, AI Assistant, Focus Planner, Performance Timer, local data, and Supabase sync.
- No database migration is required.
