# StatusOS v4.2.2 — UX & Navigation Hotfix

## Fixed

- Artist relationship notes now save silently without rebuilding the artist profile.
- Typing in artist notes no longer loses focus or moves the cursor.
- The repeated artist save interruption is removed while notes are being entered.
- A new browser/app session opens on Mission Control.
- Refreshing during the same session restores the current page instead of jumping to Success OS.
- Navigation now writes the active page directly to the workspace state.
- Updated visible version branding to v4.2.2 Mission Control.
- Updated the service-worker cache so devices receive the repaired files.

No database migration is required.
