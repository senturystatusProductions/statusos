# StatusOS v3.4.2 - Yearly Habit Goals

## Added
- Yearly schedule option in Habit Engine.
- Default yearly target of 12.
- +1 progress button for count-based yearly goals.
- -1 correction button for accidental entries.
- Yearly progress percentage and current-year tracking.

## Example
Create “Read books” with Yearly schedule and a target of 12. Tap +1 whenever a book is completed.

## Data safety
Existing daily, weekly, and monthly habits remain compatible. Current saved data is not replaced.
