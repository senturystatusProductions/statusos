STATUSOS v0.0.2 CLOUD SYNC
==========================

WHAT IS INCLUDED
- Working Supabase email/password authentication
- Cloud loading after sign-in
- Automatic cloud saving after changes
- Local browser fallback
- Current local workspace is uploaded on first sign-in if the cloud is empty

HOW TO REPLACE YOUR PROJECT
1. Stop the local server if needed.
2. Make a backup copy of your current statusos-clone folder.
3. Extract this ZIP.
4. Open the extracted StatusOS_v0.0.2_CloudSync folder.
5. Copy every file into:
   D:\New Downloads\statusos-clone
6. Choose Replace the files in the destination.
7. Restart your local server.
8. Open http://localhost:8080/
9. Press Ctrl + Shift + R.
10. Sign in.

HOW TO TEST CLOUD SYNC
1. Add a test artist on the computer.
2. Refresh the page. It should remain.
3. Open Supabase → Table Editor → statusos_workspaces.
4. Confirm one row exists.
5. Push these files to GitHub.
6. Sign in on the iPhone with the same email.
7. The same workspace should load.
