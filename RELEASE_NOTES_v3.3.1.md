# StatusOS v3.3.1 — Artist Quick Log & Contact Tracker

## Added
- One-box natural-language artist activity logging
- Automatic artist detection or creation
- Contact channel, reply, interest, and opportunity detection
- Original notes preserved exactly in the relationship timeline
- Contacted Today, Waiting for Reply, Replied, Interested, and Follow Up views
- Optional automatic follow-up dates
- Editable relationship timeline entries

## Stability
- Uses the existing Artist Repository and sync queue
- No database migration required
- Existing artists and activities remain compatible
