# StatusOS v3.9.0 — Universal Command Palette

## Added
- Ctrl/Cmd + K universal command palette
- Search across StatusOS pages, commands, and locally stored records
- Keyboard navigation with arrow keys, Enter, and Escape
- Favorites and recent commands
- Quick commands for tasks, artists, projects, revenue, focus, and workouts
- Global shortcuts: Ctrl/Cmd + N, Ctrl/Cmd + Shift + N, Ctrl/Cmd + T, Ctrl/Cmd + W

## Stability
- Preserves the v3.8.1 navigation system and page IDs
- No database migration required
