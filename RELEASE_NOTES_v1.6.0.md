# StatusOS v1.6.0 — Smart Alerts

## Added
- Focused Smart Alerts card in Mission Control
- Overdue and due-soon Music OS warnings
- Weekly, monthly, and daily habit commitment reminders
- Open-task load warning
- One-tap navigation from an alert to the relevant engine
- Calm all-clear state when nothing is urgent

## Compatibility
No Supabase migration is required. Existing tasks, habits, check-ins, projects, reviews, and settings are preserved.
